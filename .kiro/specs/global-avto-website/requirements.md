### 2.4 Система отзывов

- Пользователь оставляет отзыв без авторизации
- Поля: имя, текст, рейтинг (1-5), фото (опционально)
- Проверка по IP: 1 IP = 1 отзыв
- Отзыв уходит на модерацию (approved = false по умолчанию)
- Отображаются только одобренные отзывы

---

### 2.5 Формы заявок

- Поля: Имя, Телефон, Комментарий, Марка (авто), Модель
- Honeypot поле для защиты от ботов
- Rate limiting: не более 5 заявок с одного IP в час
- Валидация на фронте и бэке

## 4. База данных

```sql
-- Заявки
CREATE TABLE requests (
  id SERIAL PRIMARY KEY,
  name VARCHAR(100),
  phone VARCHAR(20),
  message TEXT,
  brand VARCHAR(100),
  model VARCHAR(100),
  created_at TIMESTAMP DEFAULT NOW()
);

-- Марки
CREATE TABLE brands (
  id SERIAL PRIMARY KEY,
  name VARCHAR(100),
  slug VARCHAR(100) UNIQUE,
  logo VARCHAR(255)
);

-- Модели
CREATE TABLE models (
  id SERIAL PRIMARY KEY,
  brand_id INTEGER REFERENCES brands(id),
  name VARCHAR(100)
);

-- Отзывы
CREATE TABLE reviews (
  id SERIAL PRIMARY KEY,
  name VARCHAR(100),
  text TEXT,
  photo VARCHAR(255),
  rating INTEGER CHECK (rating >= 1 AND rating <= 5),
  ip_address VARCHAR(45),
  created_at TIMESTAMP DEFAULT NOW(),
  approved BOOLEAN DEFAULT FALSE
);
```
